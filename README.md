# Queasy
Queasy est un module de quiz, ou questions / réponses.

Il propose pour l'instant une base de code pour les développeurs.

Il permet de créer des questions, et permet à chaque utilisateur d'y répondre.

Un utilisateur peut enregistrer une seule réponse par question, celui-ci ayant le choix de modifier sa réponse ultérieurement.

Parmi les types de question, il est possible de choisir entre :
`champ texte`
`textarea`
`select`
`checkboxes`
`boutons radio`

Les questions peuvent être rangés dans des groupes de question.
Récursivement, on peut créer des sous groupes de groupe.

# Prérequis
Queasy s'appuie sur le module [ACF](http://www.advancedcustomfields.com/) pour administrer les champs relatif aux groupes et questions.
